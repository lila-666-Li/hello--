# 报销搭子（Expense Advisor） v2.0

## 概述

根据用户的费用描述，智能匹配最合适的**报销科目**和**费用说明**，解决"这个该报什么科目"的难题。

**数据规模**：190条分类 × 73个科目大类，覆盖全量报销场景。
**匹配引擎**：三级分词（全词/双字/单字）+ 场景预判 + 权重评分

## 知识来源

数据文件：`expense-categories.json`（同级目录），包含190条报销科目分类。

## 使用方法

### 方式1：脚本直接查（推荐）

```bash
python3 match.py "我买了键盘和鼠标"
python3 match.py "出差打车" --top 5
python3 match.py --interactive  # 交互模式
```

### 方式2：AI 智能判断

当脚本匹配结果不明确时，结合以下规则进行判断：

#### 匹配流程
1. 从用户输入中提取**核心名词**（买了什么）和**场景词**（出差、加班、招待等）
2. 将关键词与 `expense-categories.json` 中的 `keywords` 和 `description` 字段进行模糊匹配
3. 返回匹配度最高的1-3条推荐

#### 常见场景速查
| 场景 | 推荐科目 |
|------|---------|
| 出差产生的费用（住宿/交通/餐饮） | 差旅费 |
| 办公室日常采购（文具/日用品/桶装水） | 办公用品 |
| 游戏推广/宣传/玩家活动 | 业务宣传费 |
| 请客户吃饭/送礼 | 招待费 |
| 员工聚餐/体检/福利品 | 员工福利_餐费/体检费/福利品 |
| 打车（加班或外勤） | 市内交通费 |
| 快递/邮寄 | 邮寄费 |
| 软件会员/WPS会员 | 办公费_软件服务费 |
| 酒店/租房 | 租金 |

## 注意事项

1. **成本中心**：不同部门的报销科目可能有差异，建议结合成本中心判断
2. **金额限制**：大额报销可能需要额外审批
3. **发票要求**：部分科目需要特定类型发票（如差旅费需要车票/机票）
4. **原创性**：费用说明不要照搬模板，应根据实际情况调整
5. **附件完整性**：缺少必要附件可能导致退单，提交前请对照清单检查
