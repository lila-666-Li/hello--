# 报销搭子（Expense Advisor）

> 一句话告诉AI：我该报什么科目？

## 🎯 项目背景

**痛点**：员工报销时经常不知道该选什么科目、费用说明怎么写，反复询问财务，浪费时间。

**解决方案**：输入大白话描述费用，AI秒级返回推荐科目+费用说明+关键词参考。

## 📊 提效数据

| 指标 | 使用前 | 使用后 | 提升 |
|------|--------|--------|------|
| 确认科目耗时 | 2-5分钟（查表格/问财务） | 5秒（AI自动匹配） | **96%** |
| 财务答疑频次 | 日均30+次 | 降至5次以内 | **83%** |
| 退单率（科目错误） | ~15% | <3% | **80%** |
| 覆盖科目数 | - | 73个科目，190条分类 | 全量覆盖 |

## 🛠️ 技术架构

```
用户输入（大白话）
    ↓
三级分词引擎（全词/双字/单字）
    ↓
场景预判（出差/加班/招待/游戏/办公）
    ↓
权重评分匹配（描述匹配20分 + 场景加成8分 + 关键词10分）
    ↓
Top-3 推荐结果
```

## 🚀 使用示例

### 基础查询
```bash
python3 match.py "我买了键盘和鼠标"
```

### 交互模式
```bash
python3 match.py --interactive
# 输入：出差打车 → 输出：差旅费/打车费
# 输入：请客户吃饭 → 输出：招待费/招待餐费
```

### AI 对话集成
直接在飞书对话中告诉来财：
> "我今天出差打车花了200，该报什么科目？"

来财自动调用匹配引擎返回结果。

## 📎 报销附件清单

### 付款申请-非业务款
1. 盖章版合约
2. 请款单/报价单/验收单
3. 其他（如会议签到表等）

### 费用报销
| 类型 | 必备附件 |
|------|---------|
| 交通费 | 外勤申请单、网约车行程单（如有）、所有付款截图 |
| 差旅费 | 出差申请单、网约车行程单/酒店水单（如有）、所有付款截图 |
| 其他费用 | 付款截图、其他附件（活动资料、人员清单分摊表等） |
