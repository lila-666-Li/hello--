# 报销搭子（Expense Advisor） v2.0

## 概述

根据用户的费用描述，智能匹配最合适的**报销科目**和**费用说明**，解决"这个该报什么科目"的难题。

**数据规模**：190条分类 × 73个科目大类，覆盖全量报销场景。
**匹配引擎**：三级分词（全词/双字/单字）+ 场景预判 + 权重评分

## 知识来源

数据文件：`expense-categories.json`（同级目录），包含190条报销科目分类。

## 使用方法

### 方式1：脚本直接查（推荐）

```bash
python3 match.py "我买了键盘和鼠标"
python3 match.py "出差打车" --top 5
python3 match.py --interactive  # 交互模式
```

### 方式2：AI 智能判断

当脚本匹配结果不明确时，结合以下规则进行判断：

#### 匹配流程
1. 从用户输入中提取**核心名词**（买了什么）和**场景词**（出差、加班、招待等）
2. 将关键词与 `expense-categories.json` 中的 `keywords` 和 `description` 字段进行模糊匹配
3. 返回匹配度最高的1-3条推荐

#### 常见场景速查
| 场景 | 推荐科目 |
|------|---------|
| 出差产生的费用（住宿/交通/餐饮） | 差旅费 |
| 办公室日常采购（文具/日用品/桶装水） | 办公用品 |
| 游戏推广/宣传/玩家活动 | 业务宣传费 |
| 请客户吃饭/送礼 | 招待费 |
| 员工聚餐/体检/福利品 | 员工福利_餐费/体检费/福利品 |
| 打车（加班或外勤） | 市内交通费 |
| 快递/邮寄 | 邮寄费 |
| 软件会员/WPS会员 | 办公费_软件服务费 |
| 酒店/租房 | 租金 |

## 注意事项

1. **成本中心**：不同部门的报销科目可能有差异，建议结合成本中心判断
2. **金额限制**：大额报销可能需要额外审批
3. **发票要求**：部分科目需要特定类型发票（如差旅费需要车票/机票）
4. **原创性**：费用说明不要照搬模板，应根据实际情况调整
5. **附件完整性**：缺少必要附件可能导致退单，提交前请对照清单检查
6. **服务器/云服务续费**：不走报销，走付款申请流程，需提前联系财务同事确认
7. **软件/数据库/授权类**：
   - 金额 **5w 以下** → 办公费（软件服务费/软件使用权）
   - 金额 **5w 以上** 且需要摊销 → 无形资产

## 💬 对话规则

### 何时需要反问（触发条件）

出现以下任一情况时，**必须反问**，不要直接给结果：

1. **匹配度低于 50 分** → 结果不可靠，需要反问
2. **脚本返回 ❌ 未找到匹配** → 需要反问补充信息
3. **多义场景**：
   - 「笔记本」→ 可能是文具或电脑 → 问：「请问是文具笔记本还是笔记本电脑？」
   - 「会员」→ 可能是软件会员或协会会费 → 问：「是什么类型的会员？」
   - 「聚餐」→ 可能是团建费或餐费 → 问：「这次聚餐属于团建活动吗？」
   - 「送礼」→ 客户礼品（招待费）或员工福利 → 问：「送给客户还是员工？」
4. **金额相关**：涉及软件/数据库/授权/服务器购买 → 问：「请问金额大概多少？5w 以下走办公费，5w 以上且需摊销走无形资产」
5. **边界场景**（多个科目都可能，需要用户确认）：
   - **地铁卡/交通卡充值** → 问：「这是员工通勤交通补贴，还是出差期间的交通费？」
   - **年会相关** → 问：「请问是年会场地租赁（会议费），还是年会表演/抽奖/聚餐（员工福利_活动费）？」
   - **差旅期间的餐费/外卖** → 直接匹配差旅费/餐费；若金额异常大 → 问：「请问是普通工作餐还是招待客户？」
   - **保险类** → 问：「这是出差期间的旅行保险，还是公司统一购买的商业保险？」
   - **办公用品 vs 日用品** → 问：「这是办公用的文具/耗材，还是茶水间/生活用品？」

### 反问流程（最多 3 轮）

1. **第一轮**：指出不明确的地方，给出 2-3 个选项供用户选择
2. **第二轮**：如果用户回答仍不清晰，聚焦最关键区分点（如场景/用途/金额/对象）
3. **第三轮**：两次反问后仍无法判定 → **建议用户联系分管财务同事确认**

### 反问模板

**场景不明确时**：
> 「请问这笔费用是在什么场景下产生的呢？
> A. 出差相关  B. 加班相关  C. 招待客户  D. 办公采购  E. 员工福利」

**对象不明确时**：
> 「请问这个物品是给谁使用的呢？
> A. 员工内部使用  B. 送给客户/合作方」

**金额不明确时**：
> 「请问大概金额是多少呢？5w 以下通常走办公费，5w 以上且需要摊销走无形资产」

**仍无法判定时**：
> 「这个费用场景比较特殊，建议联系您的分管财务同事确认报销科目，以免退单 😊」

### 特殊场景判定规则
- **部门聚餐**：先询问申请人是否属于团建费。确认是团建 → 员工福利_活动费；不是团建 → 员工福利_餐费
- **给客户送礼**：含「客户」+「礼品/中秋/端午/春节」→ 招待费；含「员工」+「礼品/月饼/粽子」→ 员工福利_福利品
- **会员类**：含「网站/平台/SaaS/软件」→ 办公费；含「协会/学会」→ 其它/协会会费
