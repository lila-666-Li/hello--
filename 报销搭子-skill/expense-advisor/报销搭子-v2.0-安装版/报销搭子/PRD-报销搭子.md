## 一、问题陈述

财务报销是公司内部高频操作场景。员工在报销时普遍面临以下痛点：

1. **不知道该报什么科目**：公司有73个科目大类、190条分类，员工难以准确匹配
2. **费用说明不知道怎么写**：不同科目有不同的费用说明规范，员工经常写错或不规范
3. **不清楚要传哪些附件**：付款申请、差旅费、交通费等不同类型需要的附件不同
4. **开票信息找不到**：56家公司的开票资料分散在各处，查找困难

**影响范围**：全公司所有需要报销的员工，财务部门日均处理报销答疑30+次。

**现状数据**：
- 确认报销科目平均耗时：2-5分钟（查表格或问财务）
- 财务日均报销答疑频次：30+次
- 因科目错误导致的退单率：约15%

## 四、用户故事

```
用户角色：全公司员工（需要报销的人员）

【P0 - 必须】
US-01：员工输入大白话描述费用（如"我买了键盘"），系统返回推荐报销科目+费用说明
US-02：员工查询开票信息，系统提供56家公司的开票资料查询入口
US-03：员工查询报销附件清单，系统根据报销类型返回对应附件要求

【P1 - 重要】
US-04：AI助手在群聊/私聊中自动识别报销相关关键词，主动提供指引
US-05：员工使用交互模式连续查询多个费用场景

【P2 - 期望】
US-06：系统支持根据成本中心推荐不同科目
US-07：系统自动生成费用说明文本
```

## 五、需求详细说明

### 5.1 功能需求（FR）

| 需求ID | 需求描述 | 所属用户故事 | 优先级 | 验收标准 |
|--------|----------|--------------|--------|----------|
| FR-01 | 三级分词引擎（全词/双字/单字）匹配用户输入 | US-01 | P0 | 输入"键盘"能匹配到"办公用品/电子设备及配件" |
| FR-02 | 场景预判（出差/加班/招待/游戏/办公）加权 | US-01 | P0 | 输入"出差打车"优先返回"差旅费"而非"市内交通费" |
| FR-03 | 权重评分匹配（描述20分+关键词10分+场景8分） | US-01 | P0 | 返回Top-3推荐，按匹配度排序 |
| FR-04 | 开票信息多维表格链接展示 | US-02 | P0 | 关键词"开票信息"触发时返回56家公司开票资料链接 |
| FR-05 | 报销附件清单速查 | US-03 | P0 | 关键词"附件清单"触发时返回对应类型附件要求 |
| FR-06 | 对话关键词自动指引 | US-04 | P1 | 群聊中出现"报销科目"等关键词时自动回复指引 |
